# Document Placeholders

## 1. Offer Letter

- `{date}` - Letter date
- `{employee_name}` - Full name of the employee
- `{address}` - Complete address of the employee
- `{job_title}` - Job title/position offered
- `{joining_date}` - Employee joining date
- `{signatory_name}` - Name of person signing the letter
- `{ref.no}` - Auto-generated reference number based on Reference Period

---

## 2. Internship Certificate / Letter

- `{date}` - Letter date
- `{ref_no}` - Reference number
- `{intern_name}` - Full name of the intern
- `{internship_duration}` - Duration of internship (e.g., "6 months")
- `{start_date}` - Internship start date (Employee Joining Date)
- `{end_date}` - Internship end date (Employee Resignation Date)
- `{intern_role}` - Intern's role/position
- `{work_area_1}` - First work area/description
- `{work_area_2}` - Second work area/description
- `{signatory_name}` - Name of authorized signatory
- `{signatory_designation}` - Designation of the signatory
- `{ref.no}` - Auto-generated reference number based on Reference Period

---

## 3. Experience Letter

- `{name}` - Full name of the candidate/employee
- `{role}` - Job role/position
- `{start_date}` - Employment start date (Employee Joining Date)
- `{end_date}` - Employment end date (Employee Resignation Date)
- `{letter_date}` - Letter date (optional, defaults to current date)
- `{address}` - Residential address (optional)
- `{phone_number}` - Phone number (optional)
- `{ref.no}` - Auto-generated reference number based on Reference Period
- `{signatory_name}` - Name of person signing the letter (optional)

---

## 4. Relieving Letter

- `{date}` - Letter date
- `{ref_no}` - Reference number
- `{employee_name}` - Full name of the employee
- `{resignation_date}` - Date when employee submitted resignation
- `{relieving_date}` - Date when employee is being relieved
- `{employment_start_date}` - Date when employee joined (Employee Joining Date)
- `{employee_designation}` - Employee's designation/position
- `{signatory_name}` - Name of authorized signatory
- `{signatory_designation}` - Designation of the signatory
- `{ref.no}` - Auto-generated reference number based on Reference Period
